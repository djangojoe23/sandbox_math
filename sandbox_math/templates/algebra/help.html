You must choose a step type in every step. Select the appropriate option from each step's dropdown menu. Here is what each option means:

Define 
The 'define' step type needs to be the first step type in every problem. You need to start by defining the equation to be solved. The equation must include one or two variables, which you can choose to solve for.
Rewrite

The 'rewrite' step type should be used if you are not changing the value of either side of the equation. You might be combining like terms, distributing multiplication through parentheses, or simplifying one or both sides of the equation.

For example, if the equation is 5x + 3 + 6 = 29, you would use a rewrite step for the equation to become 5x + 9 = 29. See the example if you're still not sure.

Arithmetic

The 'arithmetic' step type should be used when you are changing the value of the left and right sides of the equation. To do this step correctly, you must copy the previous step's equation exactly and then add, subtract, multiply, or divide both sides by the same value.

For example, if the equation is 5x + 9 = 29, you would use an arithmetic step to subtract 9 from both sides. Your arithmetic step should look like 5x + 9 - 9 = 20 - 9. See the example if you're still not sure.

Delete
Delete steps you no longer need. After you select Delete, you will be asked to confirm. Deleting a step cannot be undone.

After defining your equation and before adding a new step, choose the variable you will be solving for. If you don't choose a variable to solve for you will not be able to complete the problem.

There is a small "Solve for" dropdown menu below step 1. Before you define your equation this menu will be blank. After you have defined your equation, any variables in your equation will become options to choose in this "Solve for" dropdown menu.

Click the "New Step" button below the last step to create a blank new step there. It won't work if either side of the last step's equation is blank. You can always delete any steps you end up not needing by using the "Delete" step type.

It is also important to know that you should fix all mistakes before adding a new step. If there are any mistakes in any previous steps when the "New Step" button is clicked, those mistakes will be logged.

If you need help finding a mistake at a certain step, click on the icon in the expression input box on either side of the equation for that step.

You should not use these unless you do not know what else to do. Every time you click on a for help, the click is logged. As you get better, you should be clicking on these less and be able to find any mistakes on your own without the help of the computer.

Common types of mistakes are:

Define the Equation 
Use the dropdown in the first step to define an equation.
Incorrect Rewrite 
The expression is not equivalent to the previous step.
No Variable Selected 
Use the dropdown menu to select which variable to solve for.
No Step Type Selected 
Use the dropdown above this step to select which step type you want here.
Unequal Arithmetic 
The arithmetic is not the same on both sides of the equation.
Copy Previous Step 
Please copy the previous step's expression and then do arithmetic to it.

After completing a "Rewrite" step and before clicking the "New Step" button, click the "Check Rewrite" button beneath any side of an equation where you are not sure if you rewrote the step correctly.

This will start a dialog in the calculator on the right side of the screen. That will lead you through the process of checking if two expressions are equivalent by substituting any value you choose in for the variable. Type and enter "stop" in the calculator if you ever want to end the check process early. Also, if you change anything about the steps you are checking, the process will automatically be cancelled.

You can check a rewrite as many times as you want with different values for the variable each time. The number next to "Check Rewrite" on the button will increase each time you successfully do this.

However, if you check a rewrite step and find that the rewrite is not equivalent to the previous step, you found a mistake. Nice job! The number on the Check Rewrite button will turn red indicating that you found a mistake in this expression.

You should be able to remember the process of checking if two expressions are equivalent, so you can do it all on your own!

You'll know that you've isolated the variable you are solving for if the "Check Solution" button appears above the "New Step" button. This will always happen as a result of a "Rewrite" step.

When the variable you are solving for is isolated, it is by itself on one side of the equation and it does not appear on the other side of the equation. It does not matter which side of the equation the variable is isolated on.

Click the "Check Solution" button to solve and finish a problem. The "Check Solution" button appears after you have isolated the variable you are solving for.

After clicking the "Check Solution" button, a dialog will start on the right side of the screen in the calculator. This dialog will be similar to what happens when you click the "Check Rewrite" button except you will take what your variable equals in your last step and substitute it in to both sides of your starting equation.

This "Check Solution" process will help you determine if you found the right answer and if you have any mistakes in your algebra. To have the problem considered to be solved, you need the correct answer and no mistakes in your algebra.

Not showing all of your work even if you have the correct answer will count as a mistake. For example, let's say I define the equation x+5=7 and I am solving for x. I can find the value of x in my head, so I skip the arithmetic step and make my second step a rewrite step where I enter x=2. I click the "Check Solution" button and determine that I have found the correct answer, but the problem is not yet solved because there are mistakes in my algebra. I need the arithmetic step where I subtract 2 from both sides to show all my work and mark the problem as solved.
